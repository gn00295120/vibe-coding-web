在提問之前，請先查看我們的常見問題：

**[FAQ Page / 常見問題頁面](https://vibe.lucas-futures.com/faq)**

如需回報問題、建議功能或諮詢：

| Bug Report / 問題回報 | [Submit / 提交](../../issues/new?template=bug_report.md) |
| Feature Request / 功能建議 | [Submit / 提交](../../issues/new?template=feature_request.md) |

## Response Time / 回應時間

We aim to respond within **3-5 business days**.

我們會盡力在 **3-5 個工作日**內回覆。

---

## What We Cannot Help With / 我們無法協助的事項

- Payment refunds (all payments are non-refundable)
  退款（所有付款均不退款）
- Guaranteed task completion
  保證任務完成
- Private repository support
  私有儲存庫支援
